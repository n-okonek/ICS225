Homepage Design Concepts

Main Concepts:

The main idea was to bring the visitor to a very basic landing page. With all of the technical work going in behind the site, mostly server side, I wanted to give the visitor a clean landing so they are able to choose from very limited options to get where they might want to go. Since the main idea behind the site was driven by my passion for radio, and a need to "stick it to the man", I decided to take a cue from the movie Pirate Radio and go with an actual pirate themed internet radio station.

Element Concepts:

The menu is laid out in a styled unordered list to give accessibility to screen readers.

To give some visual flair, the flag will only be animated in full view. As the screen gets smaller, the script animating will not run and a static version will take its place. This will also ensure that users on mobile devices are not left hanging as the animation eats away at their RAM.

To increase contrast on the somewhat already dark background, I added a white text shadow around the text.
